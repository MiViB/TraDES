You can now include some post-translationally modified amino acids in your protein sequence.
For example, a peptide containing 4-hydroxyproline would look like: MGLGG[P:hy_g]ASIV.

These modified residues can be viewed in Cn3D - you must use the file bstdt.val that comes with the TraDES package.
